Warum Formulierung alles verändert

Rollen-Prompts fokussieren das Modell

"Erzähl mir eine Geschichte"
Gestreut — viele Bereiche
aktivieren sich schwach
vs.
"Du bist Patentanwalt.
Formuliere Anspruch 1..."
Fokussiert — relevante Bereiche
aktivieren sich stark
→ Rollen-Prompts verschieben die Wahrscheinlichkeitsverteilung in Richtung Fachsprache und -wissen.
Technik 1: Zero-Shot

Einfach fragen — ohne Beispiele

Prompt
Fasse den folgenden Patentanspruch in einfacher Sprache zusammen:

"Vorrichtung zum drahtlosen Laden eines mobilen Endgeräts, umfassend eine Sendespule, eine Empfangsspule und eine Steuereinheit, die dazu eingerichtet ist, eine adaptive Frequenzregelung durchzuführen."
✓ Schnell & einfach ✓ Gut für klare Aufgaben ✗ Unvorhersehbares Format ✗ Qualität schwankt
Technik 2: Few-Shot

Beispiele geben → Muster vorgeben

Beispiel 1
Eingabe: Vorrichtung zum Erhitzen von Flüssigkeiten...
Zusammenfassung: Ein Gerät, das Flüssigkeiten mit einem Heizelement erwärmt und die Temperatur automatisch regelt.
Beispiel 2
Eingabe: Verfahren zur Herstellung eines Verbundwerkstoffs...
Zusammenfassung: Ein Verfahren, bei dem zwei Materialien unter Druck und Hitze zu einem Verbund zusammengefügt werden.
Jetzt du
Eingabe: Vorrichtung zum drahtlosen Laden...
Zusammenfassung: → KI folgt dem Muster automatisch!
✓ Konsistentes Format ✓ Deutlich bessere Qualität ✓ KI "versteht" die Erwartung
Technik 3: Chain-of-Thought

"Denke Schritt für Schritt"

Schritt 1
Merkmale identifizieren
Extrahiere alle Merkmale aus Anspruch 1 und der Druckschrift einzeln.
→
Schritt 2
Einzeln vergleichen
Prüfe für jedes Merkmal: offenbart, teilweise offenbart, oder nicht offenbart?
→
Schritt 3
Ergebnis ableiten
Ist der Anspruch neuheitsschädlich vorweggenommen? Begründe basierend auf Schritt 2.
⚡ Zwingt das Modell zu sequenziellem Reasoning statt einer vorschnellen Antwort. Fehler werden sichtbar und nachvollziehbar.
Technik 4: Step-Back Prompting

Erst zurücktreten, dann analysieren

Deine eigentliche Frage
"Ist Anspruch 1 des Patents DE ... durch die Druckschrift EP ... neuheitsschädlich vorweggenommen?"
↑ Erst einen Schritt zurück
Step-Back Frage
"Was sind die allgemeinen Prinzipien zur Beurteilung der Neuheit eines Patentanspruchs nach EPÜ Art. 54?"
↓ Dann mit diesem Wissen
Bessere Antwort
KI antwortet jetzt mit fundiertem Verständnis der Neuheitsprüfung — nicht nur Pattern-Matching.
Der Unterschied

Was Kontext ausmacht

✗ Ohne Technik

"Schreib mir einen Patentanspruch für eine Erfindung zum drahtlosen Laden."

✓ Mit Baukasten

Du bist ein erfahrener Patentanwalt (EP/DE).
Gegeben: Sendespule, Empfangsspule, Steuereinheit mit adaptiver Frequenzregelung.
Formuliere einen unabhängigen Vorrichtungsanspruch.
Verwende EPÜ-konforme Sprache. Gliedere in Oberbegriff und kennzeichnenden Teil.

→ Gleiche Aufgabe. Völlig anderes Ergebnis. Der Unterschied sind 30 Sekunden mehr Nachdenken beim Prompt.

Weitere Techniken

Drei Bonus-Techniken

Structured Output
Fordere ein bestimmtes Format: JSON, Tabelle, Markdown. Besonders nützlich wenn du die Ausgabe weiterverarbeiten willst.
"Antworte als Tabelle mit Spalten: Merkmal | Offenbart | Begründung"
Prompt Chaining
Zerlege komplexe Aufgaben in mehrere Prompts. Ergebnis von Prompt 1 wird Input für Prompt 2.
Prompt 1: Merkmale extrahieren → Prompt 2: Neuheit prüfen → Prompt 3: Formulierung
Self-Consistency
Stelle die gleiche Frage 3x. Wenn die Antworten übereinstimmen → wahrscheinlich korrekt. Abweichungen → genauer prüfen.
3x "Ist Merkmal X offenbart?" → 2x Ja, 1x Nein → Achtung!
Praxis-Tipp

KI hat ein Kurzzeitgedächtnis

Lange Chats = verlorener Kontext = schlechtere Antworten.
